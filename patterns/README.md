# Patterns

This folder contains reusable infrastructure primitives extracted from source writing.

Patterns are not final public memos. They are generators: design grammar that can produce applied artifacts such as worksheets, rubrics, checklists, governance templates, and field-learning plans.

## Current Pattern Set

- [Mythic Infrastructure Pattern Language](mythic-infrastructure-pattern-language.md)
- [Source Index](source-index.md)
- [Prompts](prompts.md)

## How To Use

Use a pattern when an applied artifact needs a sharper mechanism.

Example:

- `Water Archive` can generate a data center cooling governance worksheet.
- `Hidden Grid` can generate a transmission/fiber/site-readiness checklist.
- `Memory Register` can generate a municipal infrastructure decision log.
- `Threshold Gate` can generate a staged review process for AI infrastructure proposals.
